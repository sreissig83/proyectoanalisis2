El usuario con Rol DEPOSITO sera el que lleve a acabo la carga de los movimientos.

Se podra agregar nuevos movinuevos pulsando el icono con el signo +.

Se debera ingresar la Fecha del movimiento, El usuario sera el que tenga la sesion iniciada y no se podra modificar esto.

Se debera seleccionar el origen del movimiento para que se habilite la opcion de seleccionar el destino.

Al habilitarse el destino, se podra seleccionar los posibles, excluyendo el origen.

Para agregar articulos al movimiento, seleccionar el boton de Agregar producto.

Se debera seleccionar de la lista el producto a cargar junto a la cantidad, si se desea agregar mas debera seleccionar el boton de Agregar producto.

Los movimientos permiten un maximo de 5 articulos en cada movimiento. Con un alerta indicando que se ha llegado al maximo.

Para registrar el movimiento se debera seleccionar el boton Aceptar.
